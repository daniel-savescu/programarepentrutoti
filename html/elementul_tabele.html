Elementele `<table>`, `<tr>`, `<th>`, `<td>` din HTML5

Tabelele HTML sunt utilizate pentru a afișa date tabulare, adică date organizate în rânduri și coloane. Sunt ideale pentru prezentarea de date structurate, cum ar fi statistici, liste de produse, orare etc.

Elementul `<table>`

Elementul `<table>` este containerul principal al unui tabel. Toate celelalte elemente ale tabelului sunt imbricate în interiorul său.

Elementul `<tr>` (Table Row)

Elementul `<tr>` definește un rând în tabel. Fiecare rând conține una sau mai multe celule de antet (`<th>`) sau celule de date (`<td>`).

Elementul `<th>` (Table Header)

Elementul `<th>` definește o celulă de antet. Acestea conțin de obicei titluri pentru coloane sau rânduri și sunt afișate în mod implicit ca text îngroșat și centrat.

Elementul `<td>` (Table Data)

Elementul `<td>` definește o celulă de date standard în tabel. Acestea conțin datele efective ale tabelului.

Sintaxa de bază:
Cod HTML5 

<table>
    <tr>
        <th>Antet 1</th>
        <th>Antet 2</th>
    </tr>
    <tr>
        <td>Dată 1</td>
        <td>Dată 2</td>
    </tr>
</table>

Structura Tabelului: `<thead>`, `<tbody>`, `<tfoot>`

Pentru o structură semantică și o stilizare mai bună, tabelele pot fi împărțite în trei secțiuni:

  • `<thead>`: Grupează rândurile de antet (de obicei conține un `<tr>` cu `<th>`-uri).
  • `<tbody>`: Grupează rândurile de date ale tabelului (conține majoritatea `<tr>` cu `<td>`-uri).
  • `<tfoot>`: Grupează rândurile de subsol (utilizate pentru totaluri, note etc.).
Exemplu de tabel complet:
Cod HTML5 

<table class="table table-bordered table-striped">
    <caption>Date despre produse</caption>
    <thead>
        <tr>
            <th>Produs</th>
            <th>Preț</th>
            <th>Cantitate</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Laptop</td>
            <td>1200 RON</td>
            <td>5</td>
        </tr>
        <tr>
            <td>Mouse</td>
            <td>50 RON</td>
            <td>20</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td colspan="2">Total produse:</td>
            <td>25</td>
        </tr>
    </tfoot>>
</table>
Atribute pentru celule:
  • `colspan`: Permite unei celule să se întindă pe mai multe coloane.
  • `rowspan`: Permite unei celule să se întindă pe mai multe rânduri.
Cod HTML5 

<table border="1">
    <tr>
        <th colspan="2">Informații Generale</th>
    </tr>
    <tr>
        <td rowspan="2">Nume</td>
        <td>Prenume</td>
    </tr>
    <tr>
        <td>Vârstă</td>
    </tr>
</table>
Stilizarea tabelelor cu CSS:

Tabelele necesită adesea stilizare CSS pentru a arăta bine. Proprietăți comune includ `border`, `padding`, `text-align`, `background-color`.

Cod CSS3 

/* CSS */
table {
    width: 100%;
    border-collapse: collapse; /* Înlătură spațiul dintre celule */
}

th, td {
    border: 1px solid #ddd;
    padding: 8px;
    text-align: left;
}

th {
    background-color: #f2f2f2;
}

tr:nth-child(even) {
    background-color: #f2f2f2; /* Rânduri alternate */
}

Utilizarea corectă a tabelelor HTML este importantă pentru a prezenta date structurate într-un mod accesibil și ușor de înțeles.
