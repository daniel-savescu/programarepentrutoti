Elementele `<ul>`, `<ol>` și `<li>` din HTML5

HTML oferă elemente dedicate pentru crearea listelor, care ajută la organizarea informațiilor într-un format structurat și ușor de citit. Există două tipuri principale de liste: liste neordonate (`<ul>`) și liste ordonate (`<ol>`). Fiecare element din aceste liste este definit de un element `<li>` (list item).

Lista Neordonată (`<ul>`)

O listă neordonată este utilizată pentru a afișa o colecție de elemente unde ordinea nu este importantă. Elementele sunt marcate de obicei cu un glonț (bullet point).

Cod HTML5 

<h3>Ingrediente pentru o salată:</h3>
<ul>
    <li>Salată verde</li>
    <li>Roșii</li>
    <li>Castraveți</li>
    <li>Ulei de măsline</li>
</ul>

Lista Ordonată (`<ol>`)

O listă ordonată este utilizată atunci când ordinea elementelor este importantă (de exemplu, pași într-o rețetă, instrucțiuni). Elementele sunt marcate de obicei cu numere, litere sau cifre romane.

Atribute comune pentru `<ol>`:
  • `type`: Definește tipul marcatorului (`1` pentru numere, `a` pentru litere mici, `A` pentru litere mari, `i` pentru cifre romane mici, `I` pentru cifre romane mari).
  • `start`: Specifică de la ce număr/literă să înceapă lista.
  • `reversed`: Inversează ordinea numerotării (de la mare la mic).
Cod HTML5 

<h3>Pași pentru prepararea cafelei:</h3>
<ol>
    <li>Pune apă în ibric.</li>
    <li>Adaugă cafea măcinată.</li>
    <li>Fierbe până se ridică spuma.</li>
    <li>Servește.</li>
</ol>

<h3>Pași cu numerotare specifică:</h3>
<ol type="a" start="3">
    <li>Pasul C</li>
    <li>Pasul D</li>
</ol>

Elementul `<li>` (List Item)

`<li>` este un element copil obligatoriu atât pentru `<ul>` cât și pentru `<ol>`. Fiecare `<li>` reprezintă un element individual în listă.

Liste Imbricate

Listele pot fi imbricate una în alta pentru a crea structuri mai complexe.

Cod HTML5 

<h3>Meniu:</h3>
<ul>
    <li>Feluri principale:
        <ul>
            <li>Friptură</li>
            <li>Paste</li>
        </ul>
    </li>
    <li>Desert:
        <ol>
            <li>Înghețată</li>
            <li>Tort</li>
        </ol>
    </li>
</ul>
Stilizarea Listelor cu CSS:

Proprietățile CSS comune pentru stilizarea listelor includ:

  • `list-style-type`: Schimbă marcatorul (ex. `none`, `circle`, `square`, `decimal`, `lower-alpha`).
  • `list-style-image`: Utilizează o imagine ca marcator.
  • `list-style-position`: Specifică poziția marcatorului (în interiorul sau în afara conținutului elementului `<li>`).
  • `padding-left`: Pentru a ajusta indentarea.
Cod CSS3 

/* CSS */
ul {
    list-style-type: square;
    padding-left: 25px;
}

ol {
    list-style-type: upper-roman;
}

Utilizarea corectă a elementelor de listă contribuie la o structură semantică a conținutului și la o mai bună accesibilitate.
