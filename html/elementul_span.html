Elementul `<span>` din HTML5

Elementul `<span>` este un container generic de tip inline (inline-level element) utilizat pentru a grupa o porțiune mică de conținut (cum ar fi câteva cuvinte sau o frază) în cadrul unui paragraf sau al altui element de tip bloc. La fel ca `<div>`, este un element "neutru" din punct de vedere semantic, ceea ce înseamnă că nu adaugă nicio semnificație inerentă conținutului.

Principalul său scop este de a permite aplicarea de stiluri CSS sau manipularea cu JavaScript pe o porțiune specifică de text fără a rupe fluxul textului.

Sintaxă și Utilizare

Sintaxa de bază este:

Cod HTML5 

<p>Acesta este un text cu o <span>parte importantă</span> de evidențiat.</p>
Caracteristici cheie:
  • Element de tip inline: Un `<span>` nu începe pe o linie nouă și ocupă doar lățimea necesară conținutului său.
  • Container generic: Este folosit atunci când nu există un element semantic inline mai potrivit (ex. `<strong>`, `<em>`, `<abbr>`).
  • Ideal pentru stilizare: Permite aplicarea de stiluri CSS (cum ar fi culoarea textului, dimensiunea fontului, fundalul) la o porțiune specifică de text.
  • Manipulare JavaScript: Oferă un punct de referință pentru a selecta și manipula porțiuni de text cu JavaScript.

Exemple de Utilizare

Stilizarea unei părți de text:
Cod HTML5 

<!-- HTML -->
<p>Prețul produsului este <span class="price">99.99 RON</span>.</p>
Cod CSS3 

/* CSS */
.price {
    color: green;
    font-weight: bold;
    font-size: 1.1em;
}
Utilizare cu JavaScript (fără a afecta layout-ul):
Cod HTML5 

<p>Numărul de vizitatori: <span id="visitorCount">0</span>.</p>
<script>
    document.getElementById('visitorCount').textContent = Math.floor(Math.random() * 1000);
</script>
Când să folosești `<span>` vs. alte elemente inline:

Folosește `<span>` atunci când nu există un element HTML semantic mai potrivit. De exemplu:

  • Pentru a face un text *îngroșat* din motive vizuale, folosește `<span style="font-weight: bold;">`.
  • Pentru a indica o *importanță* (semantică) a textului, folosește `<strong>`.
  • Pentru a face textul *cursiv* din motive vizuale, folosește `<span style="font-style: italic;">`.
  • Pentru a indica o *accentuare* (semantică) a textului, folosește `<em>`.

`<span>` este un element versatil, dar utilizarea sa ar trebui să fie ghidată de nevoia de stilizare sau manipulare a unei porțiuni de conținut, fără a introduce o nouă semantică.
