Elementul `<img>` din HTML5

Elementul `<img>` (de la "image") este utilizat pentru a încorpora o imagine într-un document HTML. Este un element de tip self-closing (nu are tag de închidere) și este esențial pentru a afișa conținut vizual pe paginile web.

Sintaxă și Atribute Cheie

Sintaxa de bază a elementului `<img>`:

Cod HTML5 

<img src="calea/catre/imagine.jpg" alt="Descriere imagine">
Atribute esențiale:
  • `src` (source):

    Specifică calea către fișierul imagine. Aceasta poate fi o cale relativă (din directorul proiectului) sau o cale absolută (un URL către o imagine online).

    Cod HTML5 
    
<img src="../imagini/logo.png" alt="Logo companie">
<img src="https://via.placeholder.com/150" alt="Imagine placeholder">
  • `alt` (alternative text):

    Oferă un text descriptiv al imaginii. Acest text este afișat dacă imaginea nu se poate încărca (din cauza unei erori de cale, conexiuni slabe, etc.) și este citit de cititoarele de ecran pentru utilizatorii cu deficiențe de vedere. Este crucial pentru accesibilitate și SEO.

    Cod HTML5 
    
<img src="foto.jpg" alt="O persoană zâmbind, în timpul unei ședințe de lucru">
Atribute opționale comune:
  • `width` și `height`:

    Specifică lățimea și înălțimea imaginii în pixeli. Deși pot fi setate aici, este recomandat să folosiți CSS pentru controlul dimensiunilor, pentru o mai mare flexibilitate și responsivitate.

    Cod HTML5 
    
<img src="profil.jpg" alt="Imagine profil" width="200" height="200">
  • `loading` (HTML5):

    Sugerează browserului cum să încarce imaginea. Valori comune:

    • `eager` (implicit): Încarcă imaginea imediat.
    • `lazy`: Întârzie încărcarea imaginii până când aceasta este aproape vizibilă în viewport-ul utilizatorului, îmbunătățind performanța.
    Cod HTML5 
    
<img src="galerie/imagine_mare.jpg" alt="Peisaj montan" loading="lazy">

Considerații importante

  • Optimizarea Imaginilor: Folosiți imagini optimizate pentru web (dimensiuni potrivite, formate precum WebP sau JPEG/PNG comprimate) pentru a asigura o încărcare rapidă a paginilor.
  • Responsivitate: Pentru a face imaginile să se adapteze la diferite dimensiuni de ecran, folosiți CSS (ex. `img { max-width: 100%; height: auto; }`) sau elementul `<picture>` pentru a oferi surse multiple de imagine.
  • Context Semnatic: Dacă o imagine este pur decorativă și nu adaugă informații esențiale, atributul `alt` poate fi lăsat gol (`alt=""`). Dacă imaginea este o figură sau un grafic important, ar trebui învelită în `<figure>` cu un `<figcaption>`.

Elementul `<img>` este fundamental pentru conținutul vizual, iar utilizarea corectă a atributelor, în special `src` și `alt`, este vitală pentru funcționalitatea, accesibilitatea și performanța site-ului tău.
