Elementul `<meta>` din HTML5

Elementul `<meta>` este utilizat pentru a specifica metadate despre un document HTML. Metadatele sunt informații despre date, în acest caz, informații despre pagina HTML în sine. Elementul `<meta>` este un element self-closing și este plasat întotdeauna în secțiunea `<head>` a documentului. Aceste metadate nu sunt afișate direct pe pagină, dar sunt esențiale pentru browsere, motoare de căutare și alte servicii web.

Sintaxă și Atribute Cheie

Elementul `<meta>` are două atribute principale pentru a defini metadatele:

  • `name`: Specifică numele metadatei (ex. `description`, `keywords`, `author`, `viewport`).
  • `content`: Specifică valoarea asociată cu atributul `name`.
  • `charset`: Specifică setul de caractere utilizat în document (ex. `UTF-8`). Acesta este un atribut special care nu folosește `name` și `content`.
  • `http-equiv`: Simulează un antet de răspuns HTTP (ex. `refresh`, `content-security-policy`).

Exemple de Utilizare Comune

1. Specificarea setului de caractere (`charset`):

Acesta este un meta tag fundamental, care ar trebui să fie primul element din `<head>` după `<title>` (sau chiar înainte). Asigură că browserul randează corect caracterele speciale și diacriticele.

Cod HTML5 

<meta charset="UTF-8">
2. Descrierea paginii (`description`):

Oferă o scurtă descriere a conținutului paginii. Este frecvent utilizat de motoarele de căutare pentru a afișa un fragment de text sub titlul în rezultatele căutării (SERP).

Cod HTML5 

<meta name="description" content="Tutoriale de programare pentru începători și avansați, inclusiv HTML, CSS, JavaScript și multe altele.">
3. Cuvinte cheie (`keywords` - mai puțin relevant SEO astăzi):

În trecut, era folosit pentru a indica cuvinte cheie relevante pentru pagină. Astăzi, majoritatea motoarelor de căutare ignoră acest meta tag sau îi acordă o importanță foarte mică.

Cod HTML5 

<meta name="keywords" content="HTML, CSS, JavaScript, programare, tutoriale, web development">
4. Autorul paginii (`author`):

Specifică autorul documentului.

Cod HTML5 

<meta name="author" content="Programare Pentru Toți">
5. Configurația viewport-ului (`viewport` - esențial pentru responsive design):

Extrem de important pentru designul responsiv, controlează dimensiunile și scalarea viewport-ului (zona vizibilă a paginii) pe diferite dispozitive.

Cod HTML5 

<meta name="viewport" content="width=device-width, initial-scale=1.0">
  • `width=device-width`: Setează lățimea viewport-ului la lățimea ecranului dispozitivului.
  • `initial-scale=1.0`: Setează nivelul inițial de zoom atunci când pagina este încărcată.
6. Redirectare (`http-equiv="refresh"`):

Poate fi utilizat pentru a reîmprospăta automat o pagină sau pentru a redirecționa către un alt URL după un anumit număr de secunde.

Cod HTML5 

<!-- Reîmprospătează pagina la fiecare 5 secunde -->
<meta http-equiv="refresh" content="5">

<!-- Redirecționează către un alt URL după 3 secunde -->
<meta http-equiv="refresh" content="3;url=https://www.example.com">
7. Open Graph (pentru partajarea pe rețele sociale):

Meta tag-urile Open Graph (OG) sunt utilizate de platformele de social media (Facebook, Twitter, LinkedIn etc.) pentru a afișa informații bogate atunci când o pagină este partajată.

Cod HTML5 

<meta property="og:title" content="Titlul Articolului Meu">
<meta property="og:description" content="O scurtă descriere a articolului pentru social media.">
<meta property="og:image" content="https://www.example.com/imagine.jpg">
<meta property="og:url" content="https://www.example.com/articol">

Elementele `<meta>` sunt cruciale pentru modul în care un document HTML este interpretat și interacționează cu lumea exterioară, influențând de la afișarea textului la performanța pe mobil și vizibilitatea în motoarele de căutare.
