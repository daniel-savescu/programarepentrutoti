Anatomia Tag-urilor HTML

Un element HTML este o componentă individuală a unui document HTML. Acesta este compus dintr-un tag de deschidere, atribute (opțional), conținut (opțional) și un tag de închidere.

Structura unui Tag HTML

Înțelegerea anatomiei unui tag este esențială pentru a scrie cod HTML corect și eficient.

Anatomia unui Tag 

<tagname attribute="value">Conținut</tagname>

Explicația Componentelor:

  1. Tag-ul de deschidere (`<tagname>`):

    Marchează începutul elementului. `tagname` este numele elementului (ex. `p` pentru paragraf, `h1` pentru titlu, `a` pentru link). Acesta este închis între paranteze unghiulare `< >`.

    Cod HTML5 
    
<p>Acesta este un paragraf.</p>
<h1>Acesta este un titlu.</h1>
  2. Atribute (Attribute="value"):

    Oferă informații suplimentare despre element, modificându-i comportamentul sau aspectul. Atributele sunt incluse în tag-ul de deschidere și au întotdeauna o pereche nume/valoare (ex. `href="url"`, `src="cale/imagine.jpg"`).

    Cod HTML5 
    
<a href="https://www.google.com" target="_blank">Vizitează Google</a>
<img src="imagine.jpg" alt="Descrierea imaginii">
    • `href`: Atributul pentru link-uri.
    • `target`: Definește unde se va deschide link-ul (`_blank` deschide într-o fereastră nouă).
    • `src`: Atributul pentru sursa imaginii.
    • `alt`: Text alternativ pentru imagine, important pentru accesibilitate.
  3. Conținut (Content):

    Reprezintă informația vizibilă sau datele pe care elementul le conține. Poate fi text simplu, alte elemente HTML sau o combinație a acestora.

    Cod HTML5 
    
<p>Acest <strong>text</strong> este important.</p>
  4. Tag-ul de închidere (`</tagname>`):

    Marchează sfârșitul elementului. Este similar cu tag-ul de deschidere, dar include o bară oblică (`/`) înainte de numele tag-ului. Majoritatea elementelor HTML necesită un tag de închidere.

    Cod HTML5 
    
<div>
    <p>Conținutul div-ului.</p>
</div>

Elemente Self-Closing (Empty Elements)

Unele elemente HTML nu conțin conținut și, prin urmare, nu necesită un tag de închidere. Acestea sunt cunoscute sub denumirea de elemente "self-closing" sau "empty elements".

Exemple de Elemente Self-Closing 

<img src="calea/catre/imagine.jpg" alt="Descriere imagine">
<br>
<hr>
<input type="text" placeholder="Introduceți text">

Exemplu Complet al unui Link (`<a>`)

Cod HTML5 

<a href="https://www.google.com" target="_blank" class="link-google">Vizitează site-ul</a>
  • <a>: Tag-ul de deschidere.
  • href="https://www.google.com": Atributul `href` cu valoarea URL-ului.
  • target="_blank": Atributul `target` pentru deschiderea într-o fereastră nouă.
  • class="link-google": Definește o clasă CSS.
  • Vizitează site-ul: Conținutul afișat ca text al link-ului.
  • </a>: Tag-ul de închidere.

Contextul elementelor în structură

Tag-urile HTML pot fi:

  • Blocuri (Block-level elements): Ocupă întreaga lățime a containerului (ex. `<div>`, `<p>`, `<h1>`).
  • Inline elements: Ocupă doar cât este necesar (ex. `<a>`, `<span>`).
