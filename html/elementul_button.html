Elementul `<button>` din HTML5

Elementul `<button>` este utilizat pentru a crea un buton clicabil. Butoanele sunt elemente interactive esențiale care permit utilizatorilor să declanșeze acțiuni, să trimită formulare sau să interacționeze cu elemente JavaScript.

Sintaxă și Atribute Cheie

Sintaxa de bază a elementului `<button>`:

Cod HTML5 

<button>Textul Butonului</button>
Atribute importante:
  • `type`: Specifică comportamentul butonului. Valori comune:
    • `submit` (implicit): Trimite datele unui formular.
    • `reset`: Resetează toate valorile unui formular la starea lor inițială.
    • `button`: Un buton generic care nu are un comportament implicit. Ideal pentru a declanșa funcții JavaScript.
    Cod HTML5 
    
<button type="submit">Trimite</button>
<button type="button" onclick="alert('Salut!')">Click Me</button>
  • `name`: Numele butonului, folosit atunci când butonul este parte a unui formular și datele sunt trimise.
  • `value`: Valoarea asociată butonului, trimisă împreună cu `name` atunci când formularul este trimis.
  • `disabled`: Un atribut boolean care, atunci când este prezent, dezactivează butonul, făcându-l necicabil.
    Cod HTML5 
    
<button type="submit" disabled>Trimite (Dezactivat)</button>

Diferența dintre `<button>` și `<input type="button">`

Deși ambele creează butoane, există o diferență importantă:

  • `<button>`: Permite un conținut HTML bogat (text, imagini, alte elemente inline). Textul butonului este plasat între tag-urile de deschidere și închidere. Este preferat pentru flexibilitate în stilizare.
  • `<input type="button">`: Este un element self-closing, iar textul butonului este specificat prin atributul `value`. Nu poate conține alte elemente HTML.
Cod HTML5 

<button>
    <img src="icon.png" alt="Iconiță" style="width:20px;">
    Trimite
</button>

<input type="button" value="Trimite">

Stilizarea Butoanelor cu CSS

Butoanele sunt adesea stilizate pentru a se potrivi cu designul site-ului.

Cod CSS3 

/* CSS */
button {
    background-color: #007bff;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
    cursor: pointer;
    font-size: 16px;
    transition: background-color 0.3s ease;
}

button:hover {
    background-color: #0056b3;
}

button:disabled {
    background-color: #cccccc;
    cursor: not-allowed;
}

Elementul `<button>` este fundamental pentru interacțiunea utilizatorului și oferă flexibilitate în design și funcționalitate.
