Elementele `<form>`, `<input>` și `<label>` din HTML5

Formularele HTML sunt utilizate pentru a colecta date de la utilizatori (ex. nume, adrese de email, mesaje) și a le trimite către un server. Elementele de bază pentru construirea unui formular includ `<form>` (containerul formularului), `<input>` (câmpuri de intrare) și `<label>` (etichete pentru câmpuri).

Elementul `<form>`

Elementul `<form>` definește un formular HTML, utilizat pentru colectarea datelor de intrare ale utilizatorului.

Atribute cheie pentru `<form>`:
  • `action`: Specifică URL-ul unde vor fi trimise datele formularului la submisie.
  • `method`: Definește metoda HTTP folosită pentru a trimite datele (`GET` sau `POST`).
    • `GET`: Datele sunt adăugate la URL ca parametri de interogare (vizibile, limită de lungime). Ideal pentru căutări sau date non-sensibile.
    • `POST`: Datele sunt trimise în corpul cererii HTTP (invizibile în URL, fără limită de lungime). Ideal pentru date sensibile (parole) sau volume mari de date.
Cod HTML5 

<form action="/submit-form" method="POST">
    <!-- Elemente de intrare aici -->
</form>

Elementul `<input>`

Elementul `<input>` este cel mai versatil element de formular, utilizat pentru a crea diverse controale de intrare de date. Este un element self-closing.

Atribute cheie pentru `<input>`:
  • `type`: Definește tipul controlului de intrare (ex. `text`, `password`, `email`, `number`, `checkbox`, `radio`, `submit`, `date`).
  • `name`: Numele câmpului, folosit pentru a identifica datele la trimiterea formularului către server.
  • `value`: Valoarea inițială a câmpului sau valoarea trimisă pentru anumite tipuri (ex. butoane radio/checkbox).
  • `id`: Un identificator unic pentru câmp, utilizat frecvent împreună cu `<label>`.
  • `placeholder`: Text sugestiv afișat în câmp înainte ca utilizatorul să introducă date.
  • `required`: Un atribut boolean care specifică dacă câmpul trebuie completat înainte de submisie.
Cod HTML5 

<input type="text" id="nume" name="nume" placeholder="Numele tău" required>
<input type="email" id="email" name="email">
<input type="password" id="parola" name="parola">
<input type="checkbox" id="acord" name="acord" value="da">
<input type="submit" value="Trimite Formularul">

Elementul `<label>`

Elementul `<label>` definește o etichetă pentru un element de intrare, îmbunătățind accesibilitatea. Atunci când utilizatorul dă click pe etichetă, focusul se mută pe elementul de intrare asociat.

Atribut cheie pentru `<label>`:
  • `for`: Valoarea atributului `for` din `<label>` trebuie să fie identică cu valoarea atributului `id` al elementului `<input>` pe care îl etichetează. Aceasta creează legătura semantică.
Cod HTML5 

<form action="/submit-contact" method="POST">
    <label for="numeUtilizator">Nume:</label><br>
    <input type="text" id="numeUtilizator" name="numeUtilizator" required><br><br>
    
    <label for="mesaj">Mesajul tău:</label><br>
    <textarea id="mesaj" name="mesaj" rows="4" cols="50"></textarea><br><br>

    <input type="submit" value="Trimite">
</form>
Considerații importante:
  • Accesibilitate: Utilizarea corectă a `<label>` cu atributul `for` este vitală pentru accesibilitate, ajutând utilizatorii de cititoare de ecran.
  • Validare: HTML5 oferă validare încorporată (`required`, `pattern`, `min`, `max`, `type="email"` etc.), dar validarea pe server este întotdeauna necesară pentru securitate.
  • Stilizare: Formularele pot fi stilizate extensiv cu CSS pentru a arăta atractiv și a fi ușor de utilizat.

Formularele sunt o componentă cheie a interacțiunii utilizator-web, iar înțelegerea acestor elemente de bază este esențială pentru colectarea eficientă a datelor.
