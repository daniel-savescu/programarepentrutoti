Introducere în HTML5

HTML5 (HyperText Markup Language 5) este a cincea și cea mai recentă versiune a limbajului standard pentru crearea de pagini web. Este coloana vertebrală a oricărui site web, definind structura și conținutul paginilor.

Ce este HTML5?

HTML5 nu este un limbaj de programare, ci un limbaj de marcare. Asta înseamnă că utilizează "tag-uri" pentru a defini elemente precum titluri, paragrafe, imagini, link-uri și multe altele.

  • HTML5 este standardul actual pentru dezvoltarea web.
  • Conținutul paginii este definit de tag-uri HTML.
  • Este interpretat de browsere web pentru a afișa paginile.

Structura de bază a unui document HTML5

Fiecare document HTML5 are o structură standard:

Cod HTML5 

<!DOCTYPE html>
<html lang="ro">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Titlul Paginii</title>
</head>
<body>
    <!-- Conținutul vizibil al paginii tale -->
    <h1>Bun venit pe pagina mea!</h1>
    <p>Acesta este un paragraf.</p>
</body>
</html>

Explicația elementelor cheie:

  • <!DOCTYPE html>: Declară tipul de document, asigurând că browserul randează pagina în modul standard HTML5.
  • <html lang="ro">: Elementul rădăcină al paginii, specificând limba documentului (română, în acest caz).
  • <head>: Conține metadatele documentului, invizibile pentru utilizator, cum ar fi:
    • <meta charset="UTF-8">: Definește setul de caractere, esențial pentru afișarea corectă a caracterelor speciale.
    • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Asigură responsive design pe diferite dispozitive.
    • <title>: Definește titlul paginii, afișat în tab-ul browserului.
  • <body>: Conține tot conținutul vizibil al paginii: texte, imagini, link-uri, etc.
    • <header>, <section>, <article>, <footer>: Elemente semantice noi în HTML5 care ajută la structurarea conținutului.

Cum să creezi un fișier HTML5?

  1. Salvează fișierul cu extensia .html, de exemplu, pagina.html
  2. Deschide fișierul într-un browser (de exemplu, Chrome, Firefox sau Edge).
  3. Vei vedea pagina structurată conform codului scris.

Aceasta este baza pentru orice proiect web. De aici poți începe să adaugi stiluri (CSS) și funcționalități interactive (JavaScript) pentru a crea aplicații web complexe!
