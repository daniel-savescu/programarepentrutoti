Elementul `<abbr>` din HTML5

Elementul `<abbr>` din HTML5 este folosit pentru a indica o abreviere sau un acronim. Prin utilizarea acestui element, poți oferi o descriere completă a termenului, care va apărea ca un "tooltip" (o bulă de ajutor) atunci când utilizatorul trece cu mouse-ul peste abreviere.

Sintaxă și Atributul `title`

Sintaxa de bază este simplă, iar atributul cheie este `title`:

Cod HTML5 

<abbr title="Organizația Națiunilor Unite">ONU</abbr>
Explicație:
  • `<abbr>`: Tag-ul de deschidere și închidere care înconjoară abrevierea.
  • `title="Organizația Națiunilor Unite"`: Atributul `title` conține textul complet al abrevierii sau acronimului. Acesta este textul care va fi afișat în tooltip.
  • `ONU`: Abrevierea vizibilă în document.

Exemple de Utilizare

  1. Abreviere simplă
    Cod HTML5 
    
Învață să codezi cu <abbr title="HyperText Markup Language">HTML</abbr> și <abbr title="Cascading Style Sheets">CSS</abbr>.
  2. Utilizarea cu un acronim
    Cod HTML5 
    
<abbr title="National Aeronautics and Space Administration">NASA</abbr> a lansat o nouă misiune.
Utilizări practice:
  • Crește accesibilitatea și claritatea conținutului.
  • Ajută utilizatorii care nu cunosc semnificația unei abrevieri.
  • Este util pentru optimizarea SEO, oferind motoarelor de căutare mai multe detalii despre conținut.
Stilizare:

Fiind un element inline, poate fi stilizat cu CSS. Un stil comun este cel de a adăuga o linie punctată sub abreviere și un cursor de ajutor:

Cod CSS3 

abbr {
    text-decoration: dotted underline;
    cursor: help;
}

Acest element îmbunătățește atât semantica, cât și accesibilitatea paginilor web.
