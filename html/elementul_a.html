Elementul `<a>` din HTML5

Elementul `<a>` din HTML5 (sau anchor element) este utilizat pentru a crea hyperlink-uri, care sunt legături către alte pagini web, secțiuni dintr-o pagină, fișiere pentru descărcare, adrese de e-mail sau chiar numere de telefon.

Acest element este fundamental pentru navigarea pe web.

Sintaxă și Atribute Cheie

Sintaxa de bază a elementului `<a>`:

Cod HTML5 

<a href="URL_destinație">Text afișat</a>
Atribute comune:
  • `href` (Hypertext Reference):

    Acesta este cel mai important atribut, specificând URL-ul (Uniform Resource Locator) către care duce link-ul.

    Cod HTML5 
    
<a href="https://www.google.com">Vizitează Google</a>
  • `target`:

    Definește unde se va deschide documentul legat. Valori comune:

    • `_self` (implicit): Deschide link-ul în aceeași fereastră/tab.
    • `_blank`: Deschide link-ul într-o fereastră sau tab nou.
    • `_parent`: Deschide link-ul în cadrul frame-ului părinte (dacă există).
    • `_top`: Deschide link-ul în fereastra completă a browserului (ieșind din frame-uri).
    Cod HTML5 
    
<a href="https://www.facebook.com" target="_blank">Deschide Facebook într-un tab nou</a>
  • `title`:

    Oferă informații suplimentare despre link, care apar de obicei ca un "tooltip" la trecerea mouse-ului peste link.

    Cod HTML5 
    
<a href="despre.html" title="Află mai multe despre noi">Despre noi</a>
  • `download` (HTML5):

    Indică faptul că link-ul ar trebui să inițieze o descărcare a resursei specificate în `href` și, opțional, specifică numele fișierului descărcat.

    Cod HTML5 
    
<a href="document.pdf" download="Raport_Anual.pdf">Descarcă Raportul</a>

Tipuri de Legături (Link-uri)

  1. Link-uri externe

    Către alte site-uri web.

    Cod HTML5 
    
<a href="https://www.w3schools.com/">Vizitează W3Schools</a>
  2. Link-uri interne

    Către alte pagini din același site.

    Cod HTML5 
    
<a href="despre_noi.html">Află despre noi</a>
  3. Link-uri către secțiuni (Ancore)

    Către o secțiune specifică în aceeași pagină sau într-o altă pagină.

    Necesită un atribut `id` pe elementul destinație.

    Cod HTML5 
    
<!-- Link către o secțiune din aceeași pagină -->
<a href="#sectiune1">Mergi la Secțiunea 1</a>

<!-- Destinația ancorei -->
<h2 id="sectiune1">Prima Secțiune</h2>
  4. Link-uri de email

    Pentru a deschide clientul de email al utilizatorului.

    Cod HTML5 
    
<a href="mailto:email@example.com?subject=Întrebare">Trimite-ne un email</a>
  5. Link-uri telefonice

    Pentru a iniția un apel telefonic pe dispozitive mobile.

    Cod HTML5 
    
<a href="tel:+40723456789">Sună-ne</a>
  6. Link-uri cu Imagini

    O imagine poate fi transformată într-un link prin învelirea tag-ului `<img>` cu `<a>`.

    Cod HTML5 
    
<a href="https://www.google.com">
    <img src="logo_google.png" alt="Logo Google" style="width:100px;">
</a>
  7. Link-uri Stilizate (Butoane)

    Deși un link este implicit text, poate fi stilizat cu CSS pentru a arăta ca un buton.

    Cod HTML5 
    
<a href="/pagina-interna.html" class="btn btn-primary">Mergi la pagina internă</a>

Elementul `<a>` este esențial în crearea de site-uri web, oferind flexibilitate pentru navigare, descărcare și interacțiuni diverse.
